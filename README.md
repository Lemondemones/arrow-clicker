# Arrow clicker

### Stack

- `React` + `Typescript`
- `Redux` + `Redux Toolkit`
- `Material UI`
- `ESLint` + `Prettier`
- `Vite`, `Vitest`


## Scripts

- `dev`/`start` - start dev server and open browser
- `build` - build for production
- `preview` - locally preview production build
- `test` - launch test runner
